Trying out
----------

1. install `Nix <https://nixos.org/nixpkgs/>`_. You should already have it.
2. ``git clone https://github.com/jyrimatti/renative-hs-example-todo.git``
3. ``./rnv.sh # Use 'rnproject' as project Name``
4. ``./build.sh``
5. ``./rnv.sh run -p web``
